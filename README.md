## **Installation**

**Using** _`poetry`_

```
git clone https://github.com/x404xx/DNS-Leaker.git
cd DNS-Leaker
poetry shell
poetry install
```

**Using** _`pip`_

```
git clone https://github.com/x404xx/DNS-Leaker.git
cd DNS-Leaker
virtualenv env
env/scripts/activate
pip install -r requirements.txt
```

## Usage Example

```bash
python -m dnsleaker
```
